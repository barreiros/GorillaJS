<center>GorillaJS</center>
===
GorillaJS es una herramienta que ayuda a automatizar fácilmente tareas complejas del desarrollo web.

<center>Con GorillaJS se puede:</center>
===
* Crear y gestionar Docker container's, en local y en remoto.
* Desplegar versiones en servidores remotos (deploy).
* Revertir los cambios en cualquier momento (rollback).
* Gestionar el histórico de versiones con git.
* Crear repositorios en Gitlab, Github y Bitbucket.
* Clonar repositorios remotos sin heredar el histórico.
